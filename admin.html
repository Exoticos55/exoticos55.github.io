Adicionar atividade

Preencha e clique em salvar. A atividade vai direto pro repositório no GitHub — não precisa editar nenhum arquivo manualmente.

1. Conexão com o GitHub

Seu token fica salvo só neste navegador (não é enviado pra nenhum lugar além do GitHub). Não use computador de terceiros sem limpar depois.

Gerado em github.com/settings/tokens?type=beta com permissão "Contents: Read and write" nesse repositório.

2. Dados da atividade

PDF, imagem, .py, .js, .zip, etc. Até ~1MB cada. Aparecem como anexos clicáveis no card.
